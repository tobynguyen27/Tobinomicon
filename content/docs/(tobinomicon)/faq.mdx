## Frequently Asked Questions

<Accordions type="multiple">
    <Accordion title="What mod loader do your mods use?" id="faq-mod-loader">
        - **Minecraft 1.20.1 and older:** These versions exclusively use **Fabric Loader**. -
        **Minecraft 1.20.1 and newer:** These versions exclusively use **NeoForge**.
    </Accordion>
    <Accordion title="Where can I download your mods?" id="faq-download">
        I only distribute my mods on **Modrinth** and **CurseForge**. Downloading from third-party
        sites poses a risk of malware or outdated files.
    </Accordion>
</Accordions>

### Troubleshooting

<Accordions type="multiple">
    <Accordion title="The game crashed! What should I do?" id="faq-crashed">
        Before submitting a report, please rule out common client-side issues:
        - **Check dependencies:** Ensure you have installed required libraries (e.g., Fabric API, Cloth Config).
        - **Verify version:** Confirm you are running the correct version of Minecraft for the mod file you downloaded.

        If the crash persists, please report it using the following steps:

        <Steps>
            <Step>Locate the `crash-reports` folder inside your `.minecraft` directory.</Step>
            <Step>Upload the latest `.txt` crash log to [mclo.gs](https://mclo.gs/).</Step>
            <Step>
            Submit the report via one of these channels:

            - **GitHub Issues:** Click the "Issues" link
            found in the mod's menu/page and follow issue template there.
            - **Discord:** If you prefer
            not to use GitHub, you can post the link in report channel is my Discord server.
            </Step>
        </Steps>
    </Accordion>

</Accordions>
