## Overview

Astral Generators is a Minecraft mod specially made for [Create: Astral](https://github.com/Laskyyy/Create-Astral). It aims to provide more ways for crafting items, producing energy and QoL stuff.

## Installation

This mod has been included in the modpack since v2.1.5.

## Features

### Machines

<Cards>
